# Booking-System-Flow
Booking System API Automation Framework using Postman, Newman, and Python 
Key features include:
- Authentication handling and token generation
- Create, update, retrieve, and delete booking operations
- API chaining using dynamic variables
- Environment-based configuration for scalability
- Positive and negative test scenarios
- Response validation using assertions
- Execution using Postman and Newman CLI
- HTML reporting for test execution results
- (Optional) Database validation to ensure backend consistency
